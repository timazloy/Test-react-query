# React Query Test

### Как запустить:

- npm install
- npm run start

### Используемые технологии:

- React Query (Tanstack query)
- Хуки useQuery, useMutation, useQueryClient
- json-server
- react-query-devtools

### Функционал:

- Получение данных и вывод их на страницу
- Добавление автомобилей в список
